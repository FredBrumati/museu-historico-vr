# 🏛️ Museu Histórico Interativo VR

Um ambiente imersivo em Realidade Virtual desenvolvido na Unity com o objetivo de apresentar conteúdos históricos sobre o Brasil de forma interativa e educativa.

---

## 📌 Sobre o Projeto

O **Museu Histórico Interativo VR** foi desenvolvido como projeto acadêmico com foco em educação, interatividade e tecnologia imersiva.

O projeto simula um museu virtual onde o usuário pode explorar diferentes salas temáticas contendo:

- Quadros históricos
- Descrições informativas
- Ambientes 3D interativos
- Navegação em primeira pessoa
- Compatibilidade com VR

O objetivo é transformar o aprendizado de história em uma experiência mais dinâmica e envolvente.

---

## 🎯 Funcionalidades

- ✅ Exploração livre do ambiente 3D
- ✅ Museu dividido em múltiplas salas
- ✅ Exibição de imagens históricas

- ✅ Interface informativa

## 🚀 Como Executar

### Pré-requisitos

- Unity instalada
- Versão recomendada: **Unity LTS**

### Passos

1. Clone o repositório:

```bash
git clone https://github.com/FredBrumati/museu-historico-vr.git
```

2. Abra o projeto pelo Unity Hub

3. Execute a cena principal

---

## ⚙️ Configuração recomendada do Unity

No Unity:

- Version Control → `Visible Meta Files`
- Asset Serialization → `Force Text`
